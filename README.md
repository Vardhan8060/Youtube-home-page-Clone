# About the Project

 > This project is a front-end clone of the YouTube homepage.
 > It replicates the layout and design of YouTube using only HTML and CSS.

 > I built this project as part of my learning journey by following tutorials and guidance from SuperSimpleDev (Simon Bao).


# Technologies Used

 > HTML5
 > CSS3
 > Flexbox
 > CSS Grid

3 Features

> Responsive YouTube-like layout
> Sidebar navigation
> Video grid section
> Header with search bar and icons
> Clean and structured UI


# What I Learned

 > Structuring web pages using semantic HTML
 > Creating layouts using Flexbox and Grid
 > Positioning elements precisely
 > Building responsive UI design
 > Organizing CSS for better readability


# Limitations

 > This is a static project
 > No JavaScript functionality
 > No backend or dynamic data
 > Buttons and search are non-functional
